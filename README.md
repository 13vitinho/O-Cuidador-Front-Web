
# O Cuidador

Site desenvolvido para o projeto O Cuidador


## Rodar localmente

Clone o projeto

```
  git clone https://github.com/13vitinho/O-Cuidador-Front-Web.git
```

Instale as dependências

```
  npm install 
```

Vá até o projeto

```bash
  cd projeto
```

Execute o site

```
  npm run dev
```

Execute 

```
 npm run backend
```

Basta clicar no link localhost e aproveitar o site! :)
